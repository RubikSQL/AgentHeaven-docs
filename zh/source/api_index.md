# API 参考

本节包含 AgentHeaven（ahvn）的完整 API 文档。

API 文档仅提供英文版本，因为它通过 Sphinx 与 autodoc 从源代码中的文档字符串生成。

<br/>

- **Core Package**: {doc}`api/ahvn` - AgentHeaven 主包及所有模块

- **Knowledge Base**: {doc}`api/ahvn.klbase` - 知识库管理
- **Tool**: {doc}`api/ahvn.tool` - 工具规范与实现
- **Imitator**: {doc}`api/ahvn.imitator` - 模仿器规范与实现
